The OPR for this supplement is 30 LRS/LGRM. AFMAN 23-110, Volume 6, Chapter 4 and
AFSPCSUP1 are supplemented as follows: The purpose of this supplement is to provide the supply
organization and shop code to be used for turn-in of precious metals and/or precious metals bearing
materials, and assign the Issue Exception (IEX) Code used on supply records to identify items of supply
that contain precious metals. Refer recommended changes and questions about this instruction to 30
LRS/LGRM, 306 8th Street, VAFB CA 93437-5322 using AF Form 847, Recommendation for Change of
Publications. Ensure that all records created as a result of processes prescribed in this publication are
maintained in accordance with AFMAN 33-363 Management of Records, and disposed of in accordance
with the Air Force Records Disposition Schedule (RDS) located at https://www.my.af.mil/gcss-
af61/afrims/afrims/rims.cfm
SUMMARY OF CHANGES
Deletes redundant paragraph 4.2.2.3.1. Deletes requirement to send the accountable officer semi-annual
transaction summary reports (paragraph 4.2.14.). Changed the IEX Code from “P” to “T” (paragraph
4.6.4).
4.6.4. (Added)IEX Code “T” will be assigned to item records which have a Precious Metals Indicator
Code assigned, or that contain a recoverable amount of precious metals.
4.9.1. (Added)Organization and shop code 120 PR will be used for turn-in of precious metal-bearing
materials.
4.9.7.1.1.2.(AFSPC Added) Organization and shop code 120 PR will be used.
